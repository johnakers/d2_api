# d2_api

Backend for d<sup>2</sup>

### Setup

```sh
$ bundle
# pulls gems

$ rails db:create db:migrate
# cretes a Postgres DB & migrates

$ rails db:seed
# only if you want some fake data
```

### Run test suitt with

```sh
$ bin/suite
```
